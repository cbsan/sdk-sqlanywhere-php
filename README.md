SDK building PHP extension SQL Anywhere
=======================================

The following binaries should be installed from your Unix installation disk if they are not already installed, and can be found as RPMs:

- make
- automake
- autoconf
- makeinfo
- bison
- gcc
- cpp
- glibc-devel
- kernel-headers
- flex
- php-dev

> \$ phpize
> \$ ./configure --with-sqlanywhere
> \$ make
> \$ sudo make install
